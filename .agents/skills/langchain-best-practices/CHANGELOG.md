# Changelog

## 1.0.0 — June 2026

### Initial Release

- 7 rule categories covering LangChain.js best practices
- Chat Models & Providers (models-)
- RAG & Vector Stores (rag-)
- Tools & Function Calling (tools-)
- Agents (agents-)
- Streaming & Callbacks (stream-)
- Structured Output (structured-)
- Prompts & Chains (prompts-)

### Key Patterns Documented

- OpenRouter integration for multi-provider access
- Supabase pgvector integration for RAG
- Zod schemas for all tool definitions
- `createAgent` vs LangGraph decision guide
- SSE streaming for Next.js API routes
- `withStructuredOutput()` for type-safe LLM responses
- `toolStrategy` for multiple structured outputs
- LangSmith tracing setup
